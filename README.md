## General Information
Word Ladder Solver adalah solusi untuk memecahkan permainan kata sederhana. Program akan meminta 2 masukan yaitu startWord dan endWord. Kemudian program akan mencari solusi-solusi optimal dari permainan ini dengan menggunakan algoritma UCS, Greedy BFS, dan A*.

## Setup
Unduh dan Instal JDK (https://www.oracle.com/java/technologies/downloads/#java8)


## Cara Menjalankan Program dan Menggunakan Program

1. Masuk ke folder src

2. Masukkan command berikut : 

    `javac Main.java`

    `java Main`
